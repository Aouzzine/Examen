Board Setup

Each player can setup his own board.

You can use the Premade setup by clicking the Place button.

Or you can set it up manually by clicking on each piece of your own color.

Then click on the dot to place them,
if you are not happy with it you can still change the position or switch with other pieces

If you are done with your setup click submit and let the other player set up his own board.

Game

After both players finished their own setup the game can start.

The lowest number on the piece is the highest rank.

Bombs and flags cannot be moved.

Bombs can only be diffused by the Miner.

Use your mouse to click on the piece and give it's movement or attack.

Once the flag is capture congratulate each other then the game will return to this page.

NOTE You can see each others pieces and do NOT prevent Alerts Have Fun!

Works great on Firefox 30+, Chrome 40+. IE in development
